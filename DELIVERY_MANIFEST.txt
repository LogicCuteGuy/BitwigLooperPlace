Basic Loop Recording
  1. create Loop1                # Create looper
  2. mark 0 0                   # Mark track 0
  3. record 0 0 true           # Enable recording
  4. [play audio into Bitwig]   # Record baseline
  5. record 0 0 false          # Stop recording
  6. launch 0 0 0              # Play clip loop

Overdub Pattern
  1. swap 0                     # Switch to track 1
  2. record 0 1 true           # Enable recording
  3. [add new performance]      # Record overdub
  4. record 0 1 false          # Stop recording
  5. Both tracks now loop together

Remote Control (OSC)
  oscsend localhost 9001 /looper/0/swap      # Swap track
  oscsend localhost 9001 /looper/0/launch 0 0  # Launch clip
